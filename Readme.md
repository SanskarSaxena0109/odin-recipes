

📝 Odin Recipes
A simple, multi-page recipe website built as part of The Odin Project curriculum. This project focuses on the fundamental building blocks of the web: HTML.

🚀 About the Project
The goal of this project was to create a basic recipe website that demonstrates mastery of HTML structure. The site features a homepage with links to various recipe pages, each detailing:

A high-quality image of the dish.

A brief, mouth-watering description.

An unordered list of ingredients.

An ordered list of the stepwise cooking process.

🛠️ Built With
HTML5 - The skeleton of the web.

Note: No CSS or JavaScript was used in this project to focus purely on markup structure.

💡 What I Learned
Through this project, I practiced:

1.Using Headings (<h1> through <h6>) to create a content hierarchy.

2.Implementing Unordered Lists (<ul>) for ingredients.

3.Implementing Ordered Lists (<ol>) for sequential steps.

4.Using Anchor tags (<a>) for internal page navigation.

5.Working with Images (<img>) and understanding relative file paths.
